## stock_market_news

     个股新闻的利好利空展示，后端使用Django，前端使用Angular2

        djangorestframework (3.6.3)
        mysqlclient (1.3.10)

       　　　 yum install mysql-devel
       
        node （v6.11.0）
        npm (3.10.10)
        @angular/cli: 1.2.0
